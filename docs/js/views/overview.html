Overview

This demo application has been designed to showcase the accessibility features of many of the Oracle JET components.

  • The Introduction page illustrates the use of a TabBar with each tab illustrating how to display data in different styles.

    • The Organization tab will show the data in a TreeView.
    • The Employees tab shows the data in a master/detail/detail format.
    • The Departments tab will utilize a filmstrip component to show the same information.
  • The Resources page will utilize the collapsible component to hide/show additional resources for Oracle JET, the W3C, and WCAG.

As you proceed through this Learning Path, you will be instructed to make changes to the application to enhance its accessibility. Finally at the end of the Learning Path you will be given a module explaining how to verify that the application is indeed accessible according to the WCAG 2.1 standards.
