Overview

This Oracle JET web application has been written to illustrate how to add accessibility features into an existing web application.

The Introduction Page:

The introduction page contains a tabbar with four tabs on it. Each tab illustrates different Oracle JET components and how they can be made accessible.

  • The Overview tab provides a couple of colored buttons which need to be made accessible to both color-blind and screen reader users.
  • The Organization page illustrates an Oracle JET TreeView and an associated data form.
  • The Employees tab illustrates the use of an Oracle JET table, an edit form in a dialog, and a messagebox.
  • The Departments tab illustrates the use of an Oracle JET filmstrip component, an Oracle JET DataGrid, and an Oracle JET bar chart.

The Resources Page:

The resources page provides a list of links to additional resources for Oracle JET, web Accessibility and various International accessibility laws.
